填充不足怎么办：补实质内容（小数据表 / 补充说明 / 引言），不要靠拉大间距充数。溢出怎么办：精简文字或拆到下一页。验收以 driver.mjs 实测为准。
为什么固定 A4 盒：浏览器自动流式分页会把图表从中间切断、产生半空白页。固定盒 + overflow:hidden + break-after:page，让每页所见即所得——屏幕一屏、打印一张，完全一致。
机制、规范与内容三分离：page-deck.css 管版面与翻页，report-skin.css 固化字体/字号/字色，你的内容只管说什么。

起一个新报告
①②③

① 复制骨架 + page-deck.css + report-skin.css。② 把每个 .page 换成你的内容（每页一个 <section class="page">），套标准 class。③ 跑 driver.mjs 验收到全 PASS。

三张文件的分工
▲▲▲

page-deck.css=版面盒+翻页+打印（勿改）；report-skin.css=字体/字号/字色（勿改）；你的 .html=内容+可选本页补充样式。

验收项手段
填充率 / 溢出 / 页数node driver.mjs your.html N
真实渲染自查Chrome --headless --screenshot
退出码0 达标 / 1 不达标 / 2 错误（可接 CI）
本地预览open your.html（macOS），屏幕即横向翻页
三张文件单一真相源：报告类项目复制这三件套即得统一视觉；改规范只改 report-skin.css 一处，所有报告同步生效。
常见坑：file:// 会被部分自动化工具拦截，验收走本地 http（driver 已内置）；flexcol 页的 SVG 若被纵向拉伸变形，把 viewBox 高度调到接近渲染比即可消除。
屏幕上：← → / 点击左右 / 滚轮 翻页。导出：Cmd/Ctrl+P → 另存为 PDF → 取消「页眉和页脚」 → 即得一页一张。
