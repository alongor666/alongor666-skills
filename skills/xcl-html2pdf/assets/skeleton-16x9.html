PDF 还是 PPT
▲▲

A4 纵向（page-deck.css）适合打印密排报告；16:9 演示（deck-16x9.css）适合屏幕 / 投影。两套共用 report-skin.css，视觉天然一致。

验收：node driver.mjs your.html N（填充率 / 溢出 / 页数）；加 --pdf out.pdf 留存 16:9 PDF。
屏幕上：← → 翻页（只认方向键）· 底部序号一键直达任意页 · Esc 切换全部页缩略图总览、点图进入。导出：Cmd/Ctrl+P → 另存为 PDF → 取消「页眉和页脚」 → 即得一页一张 16:9。
