# 结构诊断报告 · 输出骨架

每个区块标注【必选】或【可选·行业自适应】。必选区块不可省略；可选区块根据公司行业特征决定是否包含。

---

## 文档头 【必选】

```markdown
# {公司名}（{股票代码}）· 结构演变全景分析

> 分析框架：李继刚「商业结构」剑法
> 初次诊断：{YYYY-MM-DD}
> 数据截止：{YYYY-MM-DD}
```

---

## 每阶段结构（重复 3-5 次）

### 阶段标题 【必选】

格式：`## 第N阶段：{阶段名} — {一句话定位}（YYYY–YYYY）`

### 结构动力学分析 【必选】

```
**天（外部压力）：** {段落，具体到事件/数据}

**人（向心力）：** {段落，具体到人物/技术/决策}

**地（离心力）：** {段落，具体到产品/市场/地理扩张}
```

### 结构形态命名 【必选】

格式：`### 结构形态：{物理/几何隐喻}`

后接一段解释"为什么必须是这个形状"。

### ASCII 涡旋快照 【必选】

用 Unicode 方框字符（═ │ ┌ └ ┐ ┘ 等）绘制。
上方：外部压力（向下箭头）。中间：涡旋主体（边界+核心）。标注力量方向和关键节点。

### 关键财务数据表格 【可选·行业自适应】

当前阶段建议包含。历史阶段视数据可得性决定。
列：年份、营收、净利、毛利率/净利率、市占率等。

### 业务结构可视化 【可选·行业自适应】

利润金字塔、产能矩阵、客户集中度图、海外布局图等。
根据公司行业特征选用，不强制。
