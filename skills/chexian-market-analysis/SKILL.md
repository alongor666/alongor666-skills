# 车险市场分析

回答"这个市场值不值得做、该做什么"，不是解释市场概况。

## 六步推理链

**Step 1 — 判断市场阶段**（五选一）
- 红海价格战 / 结构性分层 / 渠道垄断 / 风险筛选 / 高速增长高波动

**Step 2 — 判断驱动变量**（三问必答）
- 增长靠什么？（新车 / 续保迁移 / 渠道扩张 / 价格战）
- 利润靠什么？（客户结构 / 系数 / 费用 / 赔付）
- 成交靠什么？（渠道 / 价格 / 服务 / 品牌）

**Step 3 — 判断竞争格局**
- 头部公司靠什么赢？中小公司靠什么活？华安的相对优势和短板？

**Step 4 — 识别结构机会**
- 低赔付高件均客群 / 高续保黏性客群 / 头部定价溢出区 / 新能源承保缺口 / 渠道结构空白

**Step 5 — 识别风险来源**（六类）
- 价格 / 赔付 / 费用失控 / 渠道依赖 / 政策变化 / 资源错配

**Step 6 — 输出市场结论**（四选一）
- 可重点进入 / 可选择进入 / 仅结构性进入 / 不宜进入

## 四维评估

| 维度 | 看什么 |
|------|--------|
| 吸引力 | 保费容量、增速、客群质量、赔付水平、渠道门槛 |
| 可得性 | 渠道是否愿合作、华安有无报价竞争力、本地资源 |
| 可盈利性 | 变动成本率可控、费用能换可持续保费、无短长期错配 |
| 可复制性 | 多城市可复制、不依赖个别渠道关系、不依赖短期窗口 |

## 华安专用检查（必查5项）

1. 是否品牌劣势导致成交折价？
2. 是否费用让利换保费的惯性？
3. 是否低价拿业务但赔付失控？
4. 是否只做交强险、商业险渗透不足？
5. 是否渠道依赖度高、缺乏客户直连？

## 省份特殊约束（按当前分析省份适配）

- **广东**：渠道强势、中介/4S/二网主导成交；珠三角与粤东西北分化显著
- **四川**：成都占全省40%+保费；川西/川南/川北差异大；货车险占比高

## 分析铁律

- 不准只看规模，不看成本
- 不准只看价格，不看成交
- 赔付率异常时必须拆解：`赔付率 = 出险频度 × 案均赔款 ÷ 件均保费`，频度驱动和案均驱动的对策完全不同

## 输出结构（8段）

1. 市场本质判断（一句话） 2. 竞争格局判断 3. 机会板块（3-5个） 4. 风险板块（3-5个） 5. 对华安的经营含义 6. 资源配置建议 7. 风险提示 8. 缺失数据
